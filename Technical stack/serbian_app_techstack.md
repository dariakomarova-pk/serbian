# Технический стек — Сербский язык (приложение)

---

## Стек

| Что | Что используем | Почему |
|-----|----------------|--------|
| Framework | Next.js 14 (App Router) | Роутинг из коробки, деплоится на Vercel |
| Язык | JavaScript (не TypeScript) | Проще для старта и работы с AI |
| Стили | Tailwind CSS | Утилитарные классы, хорошо работает с Claude |
| Контент | JSON-файлы в репозитории | Без базы данных на старте, всё в репо |
| Прогресс пользователя | localStorage | Для 2 пользователей достаточно |
| Хостинг | Vercel | Preview per branch + автодеплой из main |
| Репозиторий | GitHub | Уже есть |

---

## Структура деплоя

```
GitHub repo
├── main ──────────────────→ прод (yourapp.vercel.app)
└── feature/any-branch ───→ preview (yourapp-git-branch.vercel.app)
```

Каждый push в любую ветку → автоматически появляется preview-ссылка.  
Merge в `main` → автоматически идёт в прод.  
Ручных деплоев нет.

---

## Структура данных

Весь контент курса хранится в JSON-файлах внутри репозитория.

```
/data
  /course
    a1-topics.json        # список тем A1 с метаданными
    /topics
      01-alfabet.json     # контент темы: уроки, упражнения, словарь
      02-pozdravi.json
      ...
  /vocabulary
    a1-words.json         # все слова A1 с ударениями и переводами
  /phrases
    a1-phrases.json       # фразы A1 по категориям
  /kultura
    songs.json
    holidays.json
    stories.json
```

Прогресс пользователя (пройденные темы, результаты тестов, сохранённые слова) — в `localStorage`.

## Структура Next.js проекта

```
/app
  /course               # раздел Main Course
    /[topic]            # динамический роут для каждой темы
      /[lesson]         # динамический роут для каждого урока
  /vocabulary           # раздел Slovar'
  /phrases              # раздел Fraze
  /kultura              # раздел Kultura
  /grammar              # раздел Grammatika
  /profile              # раздел Profil
/components
  /ui                   # переиспользуемые UI-компоненты
  /exercises            # компоненты упражнений (flashcard, match, etc.)
/data                   # JSON-файлы с контентом
/public                 # статика: картинки, аудио
```
