# Архитектура приложения — Сербский язык
> Документ для разработки. Версия 1.0.

---

## Общее описание

Приложение для изучения сербского языка, ориентированное на эмигрантов, переехавших на Балканы (Сербия, Черногория, Босния). Цель — помочь освоить язык в реальном бытовом контексте: общение с соседями, поход в магазин, кафе, госорганы.

**Язык интерфейса:** русский  
**Изучаемый язык:** сербский на латинице (Serbian Latin script)  
**Целевая аудитория:** взрослые эмигранты, уровень 0 → A1 → A2+  
**Текущий охват:** уровень A1  

---

## Разделы приложения

### 1. Main Course — основной курс

Линейный путь изучения языка. Пользователь движется строго по порядку — нельзя пропустить тему или уровень, пока не завершён предыдущий.

#### Структура

```
Курс
└── Уровень (A1, A2, ...)
    └── Тема (14 тем на A1)
        ├── Урок 1
        ├── Урок 2
        ├── ...
        ├── Урок N (3–8 уроков на тему)
        └── Тест темы (15 вопросов)
```

#### Темы уровня A1 (по порядку)

| # | Тема | Примерно уроков |
|---|------|-----------------|
| 1 | Alfabet i izgovor | 3 |
| 2 | Pozdravi i upoznavanje | 4 |
| 3 | Porodica | 4 |
| 4 | Profesije | 3 |
| 5 | Brojevi i godišnja doba | 5 |
| 6 | Zemlje i nacionalnosti | 3 |
| 7 | U restoranu i kafani | 5 |
| 8 | Kretanje po gradu | 5 |
| 9 | Putovanja | 4 |
| 10 | Dom i svakodnevica | 5 |
| 11 | Slobodno vreme i hobiji | 4 |
| 12 | U prodavnici | 4 |
| 13 | Posao i škola | 4 |
| 14 | Kupovina i cene | 4 |

#### Состав каждой темы

Каждая тема обязательно включает:

**Грамматика**
- Объяснение правила (коротко, с примерами)
- Упражнения на закрепление

**Лексика**
- 15–25 новых слов по теме
- Слова с ударением (напр. `vòda`, `kàfa`)
- Картинка / иллюстрация к каждому слову

**Типы упражнений (вариативно по урокам)**
- Flashcard — слово + картинка, показать/скрыть перевод
- Multiple choice — выбрать правильный перевод из 3–4 вариантов
- Match pairs — соединить сербское слово с русским
- Tap the word — нажать на нужное слово среди набора
- Word order — собрать предложение из слов-кнопок
- Fill in the blank — вставить слово в предложение
- Translation — перевести фразу на сербский
- Listen & select — послушать и выбрать услышанное
- Dictation — написать то, что услышал
- Pronunciation / Read aloud — произнести слово или фразу вслух

**Культурная заметка** (1 раз на тему, в конце)
- Короткий текст (3–5 предложений) на сербском + перевод
- Связана с темой юнита
- Одно ключевое слово / понятие выделено
- Без упражнений — просто читаешь

**Тест темы**
- 15 вопросов
- Перемешаны из всех типов упражнений пройденной темы
- Нужно набрать минимальный балл, чтобы перейти дальше
- После теста — краткая статистика: что знаешь хорошо, что стоит повторить

### 2. Слова — словарь

Отдельный раздел, доступный в любое время (не привязан к прогрессу курса).

**Содержание:** все слова уровня A1 (~600 слов), сгруппированные по категориям.

**Важно:** у каждого слова обозначено ударение и русская транскрипция (приблизительная).

#### Навигация внутри раздела

```
Слова (список категорий)
└── Категория (emoji + название + кол-во слов)
    ├── Кнопки режима: [ Флэшкарты ] [ Найди пару ]
    └── Список слов категории
```

#### Режимы тренировки

- **Флэшкарты** — листаешь карточки: сербское слово + транскрипция + перевод. Без оценки, просто просматриваешь.
- **Найди пару** — 6 пар за раунд: нужно соединить сербское слово с русским переводом.

#### Категории словаря

| # | Категория | Слов |
|---|-----------|------|

| 4 | Профессии | ~25 |
| 5 | Числа и времена года | ~45 |
| 6 | Страны и национальности | ~30 |
| 7 | Еда и кафе | ~40 |
| 8 | Город и транспорт | ~35 |
| 9 | Путешествия | ~30 |
| 10 | Дом и быт | ~35 |

| 12 | Магазин и покупки | ~30 |
| 13 | Работа и учёба | ~30 |
| 14 | Глаголы A1 | ~50 |

### 3. Fraze — фразы

Отдельный раздел с разговорными фразами уровня A1. Доступен в любое время.

**Содержание:** ~80–100 фраз, сгруппированных по ситуациям:
- Pozdravi (приветствия)
- Upoznavanje (знакомство)
- U kafiću / restoranu (в кафе / ресторане)
- Kupovina (покупки)
- Kretanje (навигация по городу)
- Preživljavanje (фразы выживания — "Ne razumem", "Možete li sporije?")

**Режимы тренировки:**
- Собери фразу — расставить слова в правильном порядке
- Найди перевод — выбрать правильный перевод фразы
- Произнеси фразу — прочитать вслух, оценка произношения (speech recognition)

---

### 4. Kultura — культура

Раздел, где язык объясняется через культурные артефакты. Без прогресс-баров и обязательных упражнений. Читаешь в своём темпе.

**Форматы контента:**

| Формат | Описание | Примеры |
|--------|----------|---------|
| 🎵 Pesma | Разбор песни: текст, перевод построчно, история | Đorđe Balašević, народные песни |
| 🎭 Anegdota | Анекдоты и народные истории с разбором | Муйо и Хасо, пословицы |
| 🎉 Praznik | Праздники и традиции | Slava, Božić, Uskrs |
| 📖 Priča | Короткие тексты, стихи, микрорассказы | Десанка Максимович, 6-word stories |
| 🗺️ Mesto | Места, города, бытовая культура | Beograd, kafana, srpska hrana |

**Механика:**
- Каждый материал маркирован уровнем (A1 / A2)
- Новые слова из текста можно сохранить в словарь одним нажатием
- Аудио там, где есть (песни, стихи)
- Связь с темами курса: материал может быть отмечен как "связан с темой X"

**Связь с курсом:**  
В конце каждой темы курса есть короткая культурная заметка (inline). Полный материал — в разделе Kultura.

---

### 5. Grammatika — грамматика

Справочный раздел. Доступен в любое время, не привязан к прогрессу.

**Содержание:** объяснения ключевых правил сербского A1 с примерами:

- Глагол `biti` (настоящее время)
- Настоящее время — регулярные глаголы (-ati, -iti, -eti)
- Род существительных (muški, ženski, srednji)
- Падежи: введение в систему (именительный, винительный, родительный — базово)
- Притяжательные местоимения (moj, tvoj, njegov, njen...)
- Указательные местоимения (ovaj, taj, onaj)
- Модальные глаголы (moći, morati, hteti)
- Предлоги места (u, na, do, od, kod, iz)
- Порядок слов в предложении

**Формат каждого правила:**
1. Краткое объяснение (2–4 предложения)
2. Таблица / схема если нужна
3. 2–3 примера предложений с переводом
4. Ссылка на тему курса, где это правило встречается

---

### 6. Profil — профиль и настройки

Открывается через иконку ⚙️ в верхнем хедере (не вкладка таб-бара). Экран с настройками и статистикой пользователя.

**Содержит:**
- Прогресс по курсу (какие темы пройдены, текущий уровень)
- Статистика: дней подряд, слов выучено, тестов пройдено
- Настройки: уведомления, скорость аудио
- Возможность сбросить прогресс по отдельной теме

## Технические заметки

- **Язык интерфейса:** русский (все объяснения, переводы, интерфейс — на русском).
- **Изучаемый язык:** сербский исключительно на латинице (Serbian Latin). Кириллица не используется.
- **Ударения в словаре:** обозначаются знаком grave над гласной (`à`, `è`, `ì`, `ò`, `ù`) или через отдельное поле с транскрипцией.
- **Аудио:** нужно для словаря (произношение каждого слова), фраз (произношение), культуры (песни, стихи). Желательно — голос носителя.
- **Speech recognition:** используется в разделе Fraze для упражнений на произношение.
- **Уровни языка:** сейчас реализуется A1. Архитектура рассчитана на последующее добавление A2, B1.

## Навигация приложения

### Верхний хедер (на всех экранах)

```
[ 🔥 N дней ]                    [ ⚙️ ]
  streak слева             настройки справа
```

- **Streak** — количество дней подряд с активностью
- **⚙️** — открывает профиль и настройки (Profil)

### Нижний таб-бар

```
[ Курс ] [ Слова ] [ Фразы ] [ Культура ]
```

- **Grammatika** — доступна из раздела Курс (кнопка в хедере темы)
- **Профиль и настройки** — через иконку ⚙️ в верхнем хедере (не вкладка)
